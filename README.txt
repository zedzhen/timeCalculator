Для работы необходимо установить библиотеки keyboard, pystray, Pillow и requests
При запуске без модулей программа спросит попробовать ли их усстановить их автоматически

Для ручной установки
python -m pip install keyboard pystray==0.19.1 Pillow requests
(директория к python должна быть в PATH
или запускать из директории с python
или вместо python указать полный путь до него)
